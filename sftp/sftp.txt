3. Передача файлов
3.1 Передача файлов на локальную систему

get файл

get файл ДругоеИмяФайла (скопировать файл под другим именем)

get -r Директория (скопировать директорию)

get -Pr Директория (скопировать директрию с сохранением привелегий, даты и времени доступа)

3.2 Передача файлов на удаленную систему 
put файл
put -r Директория 

4. Простые операции
chown userID file (изменить владельца файла, вместо имени используется ID пользователя)

узнать ID:
get /etc/passwd
!less passwd

chgrp groupID file (изменить группу владельцев файла)
get /etc/group
!less group


chmod 777 файл (изменить права файла)
